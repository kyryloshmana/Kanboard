
*****Вибір Браузера*********
-Pfirefox - прописується для запуску тестів в браузері firefox
-PchromeHeadless - прописується для запуску тестів в браузері chromeHeadless
-Pchrome - прописується для запуску тестів в браузері chrome
*****Вибір тестів*************
-PUI - вибі UI тестів
-PCombined - вибір Combined тестів
-PAPI - вибір API тестів
*****Команди для запуску тестів****************
mvn clean test -PAPI - Запуск API тестів
mvn clean test -Pfirefox -PUI - Запуск UI тестві з браузером
mvn clean test -Pfirefox -PCombined - Запуск UI тестві з браузером
mvn clean test -PchromeHeadless -PUI - Запуск UI тестві в режимі Headless
mvn clean test -PchromeHeadless -PCombined - Запуск Combined тестві в режимі Headless
mvn clean test -Pchrome -PUI - Запуск UI тестві з браузером chrome
mvn clean test -Pchrome -PCombined - Запуск Combined тестві з браузером chrome
